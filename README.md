ข้อมูลทั่วไป
- [Citizens Information](https://www.citizensinformation.ie/) - กฎระเบียบเรื่องการเข้าเมือง ใบอนุญาตพำนัก การรักษาพยาบาล สิทธิของผู้เช่าบ้าน สิทธิแรงงาน ไม่ว่าจะหาอะไร ลองที่เว็บไซต์นี้ก่อนเป็นแห่งแรก เป็นเว็บไซต์ข้อมูลที่ดำเนินงานโดยการสนับสนุนของรัฐบาลไอร์แลนด์ ข้อมูลทุกอย่างจะมีลิงก์ไปยังหน้าเว็บหรือเอกสารที่เกี่ยวข้องของหน่วยงานรัฐที่รับผิดชอบโดยตรง - [https://www.citizensinformation.ie/](https://www.citizensinformation.ie/)

การรักษาพยาบาล / ประกันสุขภาพ
- [Medical cards](https://www2.hse.ie/services/schemes-allowances/medical-cards/) - ผู้พำนักทุกคนมีสิทธิสมัครรับบัตรรักษาพยาบาลฟรี ทั้งนี้จะพิจารณาจากรายได้และรายจ่าย ซึ่งถ้าเข้าเกณฑ์ก็จะได้รับบัตร "medical card" โดยในระหว่างกระบวนการจะมีให้เลือกหมอ GP (general practitioner)
- ก่อนซื้อประกันสุขภาพ ให้ตรวจสอบกับสถานศึกษาว่ามีแผนประกันของสถานศึกษาหรือไม่ ซึ่งเป็นไปได้ว่าจะมีราคาถูกกว่าหรือมีเงื่อนไขการประกันที่ดีกว่า
